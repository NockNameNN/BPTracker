# BP Tracker

Веб-приложение для отслеживания ежедневных заданий и Bonus Points (BP) в GTA 5 RolePlay.

## Возможности

- **Избранные задания** — выбирайте задания со страницы «Все задания» и отслеживайте прогресс на главной
- **Ежедневный сброс** — прогресс обнуляется в 07:00 по МСК
- **VIP-статус** — переключатель влияет на расчёт BP (с учётом бонуса за VIP)
- **Фильтры и сортировка** — по категории (фракционные / не фракционные), доп. условиям, времени, BP
- **Сетапы** — сохраняйте наборы избранных заданий под именами и быстро переключайтесь между сценариями (фракция, оффлайн-день и т.п.)
- **PWA** — установка на устройство и работа офлайн
